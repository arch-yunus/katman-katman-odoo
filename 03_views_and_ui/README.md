# 03 - Görünüm ve Sunum Katmanı (Views and UI Layer)

## Odak
XML tabanlı kullanıcı arayüzü (UI) bileşenleri ve kalıtım (Inheritance).

## İçerik
- **Form Görünümleri:** Temel ve gelişmiş form tasarımı
- **Tree Görünümleri:** Liste ve hiyerarşik görünümler
- **Kanban Görünümleri:** Sürükle-bırak, renkli kartlar
- **Search Görünümleri:** Filtreler ve özel alanlar
- **View Inheritance:** XPath ile mevcut görünümleri genişletme
- **Dinamik Alanlar:** attrs, domain, modifiers

## Hedef
Kullanıcı dostu, dinamik ve esnek ekranlar tasarlamak.

## Başlarken
Bu klasörde bulacaksınız:
- Form, Tree, Kanban, Search örnekleri
- View Inheritance teknikleri
- UI best practices
