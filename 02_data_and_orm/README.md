# 02 - Veri ve ORM Katmanı (Data and ORM Layer)

## Odak
Odoo Veri Modelleri (Models), Alanlar (Fields) ve Python ORM yetenekleri.

## İçerik
- **Model Türleri:** `models.Model`, `models.TransientModel`, `models.AbstractModel`
- **Alan Türleri:** Text, Integer, Float, Many2one, One2many, Many2many
- **Computed Alanlar:** @api.depends ve dinamik hesaplamalar
- **onchange Fonksiyonları:** Kullanıcı arayüzünde gerçek zamanlı güncellemeler
- **Validasyon ve Kısıtlamalar:** _sql_constraints, @api.constrains

## Hedef
İş mantığını Python tarafında hatasız ve performanslı bir şekilde kurgulamak.

## Başlarken
Bu klasörde bulacaksınız:
- Özel model örnekleri
- ORM işlemleri (CRUD)
- İlişkisel veritabanı tasarımı örnekleri
