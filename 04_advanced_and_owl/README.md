# 04 - Gelişmiş Özellikler ve OWL Framework (Advanced Features & OWL)

## Odak
Odoo Web Library (OWL), JavaScript entegrasyonu, API'ler ve optimizasyon.

## İçerik
- **OWL Bileşenleri:** Modern JavaScript/TypeScript bileşenleri
- **REST API / XML-RPC:** Harici sistemler için entegrasyon
- **Otomatik Aksiyonlar:** Tetikleyiciler ve otomasyon kuralları
- **Sipariş Yenileme Kuralları:** Reordering Rules ve çalışma kuralları
- **Performans Optimizasyonu:** Sorgu iyileştirmeleri, indeksler
- **Ek Modüller:** Raporlar, Batchler, Araçlar

## Hedef
Odoo'yu sınırlarının ötesine taşımak ve performanslı edge-integrasyonlar kurgulamak.

## Başlarken
Bu klasörde bulacaksınız:
- OWL bileşen örnekleri
- API entegrasyonları
- Otomasyon scriptleri
- Performans ipuçları
